# Pixel Divination 产品规划文档

> **版本**：v2.0  
> **更新日期**：2026-06-13  
> **产品愿景**：成为最专业、最沉浸的 AI 玄学工具平台

---

## 一、产品概述

### 1.1 产品名称
**Pixel Divination（像素占卜）**

### 1.2 产品定位
赛博朋克风格的专业级 AI 玄学平台，融合西方占星、东方命理、塔罗牌阵、雷诺曼、风水等多元占卜体系，通过大语言模型提供深度个性化解读。

### 1.3 目标用户
| 用户画像 | 特征 | 核心需求 | 占比 |
|----------|------|----------|------|
| 轻度用户 | 18-30岁，对星座感兴趣 | 每日运势、娱乐消遣 | 60% |
| 中度用户 | 了解塔罗/占星基础 | 深度解读、个人星盘 | 25% |
| 专业用户 | 玄学爱好者/从业者 | 专业工具、多体系对比 | 10% |
| 社交用户 | 愿意分享、寻求认同 | 社交货币、匹配功能 | 5% |

### 1.4 竞品对标分析

| 竞品 | 核心优势 | 我们的差异化 |
|------|----------|--------------|
| **Astrodienst** | 专业占星数据、多分宫制 | 更好的视觉体验+AI解读 |
| **Co-Star** | 社交化、极简设计 | 更多占卜体系+深度分析 |
| **Sanctuary** | AI占星师、精美UI | 赛博朋克风格+东方命理 |
| **Labyrinthos** | 塔罗学习系统 | 多体系融合+即时AI解读 |
| **测测星座** | 本土化、功能全 | 更专业的技术+AI能力 |
| **准了** | 数据准确、专业 | 更好的体验+多元体系 |

## 四、技术架构

### 4.1 当前技术栈
```
前端：React 19 + TypeScript + Vite
状态管理：Zustand
样式：Tailwind CSS v4
动画：Framer Motion
AI：OpenAI SDK → DeepSeek API
占星计算：astronomy-engine（本地）
```
